Recent Projects

Tic Tac Toe Erasure

This is a game where the user plays the computer on a grid of 3 by 3 in an attempt to have either 3 x's or 3 o's in a row or column or diagonal depending on whether they play with x's or o's respectively. The game has a twist where the board only stores 3 of the most recent plays, erasing past plays as you go.

Sudoku Solver

This is a program that solves normal sudokus, i.e. 6x6, 9x9, 16x16 sudokus. There are two ways to go about inputing the sudoku: one can either enter the sudoku size of the smaller block and enter the prefilled values or they can take a photo and use the photo as input. In the latter case, the program has AI that analyses the image gets the corners of the grid, reshapes it to a square then runs character recognition to get the values in the cells. With this information it creates an object that can be solved and returns an image of the solved sudoku.

AI Style Transfer
